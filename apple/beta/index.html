OS X预览

OS X El Capitan

每一点，都有更多喜爱的理由。

今秋推出

如今，有越来越多的人喜欢使用Mac电脑。这其中很重要的一个原因，就是 OS X 。可以说，Mac之所以是Mac， OS X 功不可没。它既能让你轻松成就非同寻常之事，也能让你愉快处理一切日常之事。 OS X El Capitan 继承了 OS X Yosemite 开创性的功能和精致设计，并在众多看似微不足道、实则影响重大的方面进行了功能和性能上的提升。 OS X El Capitan ，将Mac的使用体验带到了一个全新的高度。

体验

OS X El Capitan 让你能以更简单、更智能的方式处理在Mac上经常要做的事。例如，同时在多个app中进行操作，搜索信息，关注自己喜欢的网站，查收电子邮件，以及做笔记等等。

性能

我们对OS X的内在进行了一系列的提升，让Mac能够敏捷高效地处理从打开PDF文件，到查收电子邮件等各种日常任务。而适用于Mac的Metal技术，还让游戏、高性能app和其他许多方面的图形性能变得更快速、流畅。

Split View

谁说一心不能二用？

使用Mac的一大优势，是能同时运行多个app；而另一大优势，则是你能以全屏视图专注于单个app。有了Split View，现在你能让这两大优势一并呈现。它会将你选择的两个app自动填满整个屏幕。所以你可以一边在信息app中与好友讨论晚餐计划，一边在地图app中查找喜爱的餐厅；或者一边在Pages中处理文档，一边在Safari中查找资料。你既不会受到其他已打开app的干扰，也不必手动调整窗口大小和来回拖动窗口。而且只需轻扫一下，就能回到桌面，从而可以轻松切换到之前进行的其他事上。

Mission Control

空间管理，从未如此游刃有余。

更简洁的 Mission Control ，可让你更加轻松地查看和整理Mac上已打开的一切。只需轻扫一下，你桌面上的所有窗口便会在同一层展开，不会相互堆叠或遮挡。 Mission Control 还会将窗口按照它们在桌面上的相应位置进行排布，因此你可以更快找到想要的那个窗口。而当你有很多窗口相互挤占桌面时，现在也有了更简单的方式来给它们腾出更多空间。只需将任何一个窗口拖至屏幕顶部，就能把它放入全新的桌面空间中。展开工作，从未如此轻松。

让你的光标跃然眼前

桌面拥挤混乱，找不到光标了？这时只需在触控板上晃晃手指，或是摇摇鼠标，光标便会变大，让你可以轻松找到。

字体和文本输入

Mac现在变得更地道了

全新的系统字体和功能更强的输入法，使中文文本在Mac上看起来更加赏心悦目，输入时格外简单轻松。

全新的系统字体

全新设计的中文系统字体名为"苹方"。它具有现代感的外观和清晰易读的屏幕显示效果，并同时支持简体中文和繁体中文。

手写输入

用手指在触控板上输入文字，现在也可以像在纸上书写一样流畅自如、快速准确。全新的触控板窗口可以同比映射你在实体触控板上的操作，因此你有更多书写空间，而且可以一行书写多个字符。

输入预测

用键盘输入中文从未如此快速简单。得益于先进的学习功能，系统可快速记忆你的词语选择；提升的语言预测引擎，能更准确地预测你要输入的下一词语；定期更新的词汇库包含最新的词语和短语；而更智能的候选词窗口，可为你显示更多字符选择。

Spotlight

更多才，更好用。

有了 OS X El Capitan ，你就可以用自己的话在Spotlight中查找文件。而且，Spotlight还变得更加灵活，让你能调整窗口大小查看更多结果，还能在桌面上将它随意移动。

用你自己的话来查找

Spotlight能够理解日常语言，这让文件搜索变得前所未有的简单 1 。打个比方，你只要输入''Harrison四月份发的邮件''，Spotlight便会显示与之匹配的电子邮件。你还可以执行更复杂的搜索，例如输入''昨天我做的包含日程的文件''，就可以自己找到想要的那个。除此之外，在邮件app和Finder中你也可以同样使用自己的话来搜索。

邮件

邮箱又有新收获

邮件app增强了全屏支持和轻扫手势功能，让你可以快速处理邮件。 OS X 甚至能帮助你直接在收件箱中管理日历和更新通讯录。

全屏操作更轻松

邮件app增强了全屏视图功能，让你可以同时掌控所有的电子邮件会话。现在，你所撰写的电子邮件可以滑至屏幕底部，以便你访问收件箱，这一点非常适合在各个邮件之间复制文本或添加附件。如果你正在处理多封电子邮件，还可以通过简单易用的标签页来回切换。

添加建议的事件

当你收到的电子邮件包含航班、晚餐预订等事件的详细信息，或是像''我们5点去喝一杯''这类的提议时，只需点按一下，即可将事件添加到日历中 2 。

添加建议的联系信息

当你和不在通讯录中的联系人邮件往来时，邮件app让你只需点按一下，便能将他们添加到通讯录中 3 。当其中某个联系人的电子邮件地址发生更改时，它还会提醒你。

轻扫即可管理收件箱

现在，你可以像使用iOS设备一样通过轻扫手势管理电子邮件。需要对收件箱内容进行分拣？只需向右轻扫将邮件标记为已读或未读，或向左轻扫删除邮件即可。这样你便能第一时间关注到那些重要信息。

备忘录

记录想法的方式，何止是文字。

对于功能强大的全新备忘录app来说，它的出色之处，绝不仅仅是可让你随手记下闪现的灵感，并于日后方便取用。现在，它还能让你快速将列表变成核对清单；或者轻松向备忘录中添加照片、视频、链接或地图上的位置。而且得益于iCloud，所有备忘录和其中的内容均可在你的所有设备上保持更新。因此，你可以在Mac上创建一个备忘录，然后外出时用iPhone来查看。

从其他App收集资讯

正在计划一次旅行？你可以直接从Safari中将酒店的网站保存到备忘录，也可以直接保存地图app中的某个地址。而且，你还能从其他许多app中保存资讯到备忘录。只要点按''共享''按钮，便可将资讯保存到现有的备忘录，或创建为新的备忘录。

创建好用的清单，只需轻轻一点。

现在，你可以轻轻松松在备忘录app中创建核对清单。只需点按''制作核对清单''按钮，即可创建交互式待办事项列表、购物清单或者是愿望清单。完成清单项目后，点选勾除即可。

附件浏览器，所有附件一处尽览。

全新的附件浏览器，可将你添加到备忘录的所有附件统一整理在一个简洁的视图中。你可以轻轻松松对照片和视频，地图位置，以及网站等进行分类浏览，而无需牢记它们存在哪条备忘录中。只需要点按一个附件，即可直接转到它所在的备忘录。

你的备忘录，在你的所有设备呈现。

备忘录app可以与iCloud相互协作，因此无论你使用什么设备，你的备忘录都会保持更新并与你形影不离。比如你在Mac上创建了一份核对清单，当你外出时，它将出现在你的iPhone上。而如果你在iPhone上勾除了一个清单项，它也会从你的Mac上被勾除。你在一部设备上对备忘录所做的任何更改，都将立即呈现在你其他所有设备之上。

照片

可做更多处理，也可按地点、人物轻松整理。

有了第三方编辑工具，你可以为照片增添更具个性化的风采。加上功能更强的整理工具和更快的性能，照片app现在变得更加出色。

全新扩展编辑工具，让照片效果更进一步。

OS X El Capitan 支持第三方编辑工具。你可以从 Mac App Store 获取它们，并直接在照片app中使用。你可以用自己喜爱的开发者带来的多种工具来编辑单张照片，也可以将它们与照片app内置的编辑工具一起综合使用。从添加精妙入微的滤镜，到采用精致美观的纹理效果，你的照片编辑水准将由此提升到全新高度。

一切各就各位

优化之后的照片app，让管理照片图库变得更轻松。现在，你可以为单张照片或整个"时刻"添加位置信息；简化的操作流程，还能让你更快为''面孔''中喜爱的人物添加姓名；你还可以按日期、标题等信息对相簿及其中的内容进行排序。

Safari

智能的浏览器，更添新的内置工具。

在 OS X El Capitan 中，这款Mac上的卓越浏览器拥有了众多全新工具，可带给你更出色的上网浏览体验。你现在能借助固定标签页功能，让自己喜爱的网站在Safari中始终保持打开状态，以便随时访问。你还可以快速将页面音频静音，而无需去翻寻声音来源。

固定标签页，
把你喜爱的网站留在手边。

就像用图钉固定东西一般，你经常喜欢访问的那些网站，现在也能够以类似的方式保持打开及更新状态，让访问更方便。它们将在后台保持活动状态，并一直挂在标签栏的左侧。

轻轻松松，让页面静音。

想快速停止音乐，又不想去翻找是哪个页面在播放音乐？现在，你可以直接在智能搜索栏中将其静音。如果你正在聆听某个页面的音频，而这时有其他网站的音频响起，你可以选择将不想听的那个静音。而如果你真正想要的是一个安静的氛围，也可以选择将浏览器中的所有音频静音。

地图

公共交通信息，即将抵达。

如果你搭乘火车或公共汽车出行，那么现在地图app可让你在全球某些城市的出行更加轻松便捷。你将找到整个行程所需的一切信息，包括内置的公交线路图、路线指引，以及时刻表。

公共交通，大家乐享。

只要在''公交''视图中选择一个目的地，地图app就会给出最佳路线安排，包括有详细的步行、地铁、火车、公共汽车和渡船路线指引。你也可以查看复杂行程的路线，例如先搭乘公共汽车，之后步行两个路口再转乘地铁。还可根据自己希望的出发时间，或需要的到达时间来规划行程。

在Mac上获取路线，在路上来导航。

你可以在自己的Mac上规划好路线，然后只需点按几下便可将路线发送至iPhone，为你的出行提供逐站导航。

性能

Mac越来越机敏

在 OS X El Capitan 中，无论是查收电子邮件，还是启动或切换app，我们让各个方面的运行速度都有了进一步提升。正是这些细微之处的提升，让你感觉到自己的Mac速度更快，反应更灵敏。我们还将Metal技术纳入Mac之中，因此无论你是玩游戏，运行高性能app，还是执行关键的系统级图形操作，都将感受到更加流畅的图形性能。

Metal技术，让像素渲染全速表现。

Metal是一项全新的图形处理核心技术，它让游戏和app可以更直接地使用Mac的图形处理器，从而带来更强劲的性能表现和更丰富的图形体验。Metal技术可将系统级图形渲染速度提升最高达50% 4 ，并将效率提升最高达40%之多 5 。它还能让主处理器和图形处理器更高效地协作，加快高性能app的运行速度。Metal技术还为打造更出色的游戏体验而设计，它令绘图调用性能有了高达10倍的提升，并为强化游戏的真实感和细腻度开创了有利条件 6 。

性能全线提升

现在，你在电脑上所做的一切日常之事，如启动和切换app，打开PDF文件，查收电子邮件等等，都将变得更加快速流畅。 OS X El Capitan 会让你的Mac用起来感觉非常流畅和灵敏。

和我们一起，将 OS X El Capitan 变得尽善尽美。

Apple Beta Software Program计划，让你有机会能在 OS X El Capitan 于今年秋天正式发布前先睹为快。参加该计划，你将可以试用预发布版 OS X ，并向我们反馈你的使用感受。
